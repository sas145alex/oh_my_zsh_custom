# oh_my_zsh_custom
My personal custom settings to oh-my-zsh

## Install
```bash
cd $ZSH
git clone https://github.com/sas145alex/oh_my_zsh_custom.git custom
```

## Recommended plugins
```bash
plugins=(
  # standart plugins
  git
  gem
  docker
  colored-man-pages

  # custom plugins
  zsh-syntax-highlighting
  zsh-autosuggestions
)
```
